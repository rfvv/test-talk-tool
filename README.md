# Test Talk Tool

这是一个使用现代技术栈构建的对话工具项目。

## 技术栈

- React
- TypeScript
- Tailwind CSS

## 项目结构

```
test-talk-tool/
├── public/           # 静态资源
├── src/              # 源代码
│   └── components/   # React 组件
├── package.json      # 项目依赖
└── tailwind.config.js # Tailwind 配置
```

## 开始使用

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm start
```

3. 构建生产版本：
```bash
npm run build
```
